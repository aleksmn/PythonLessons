Установка библиотек

Windows:

python -m pip install pip — для установки/обновления пакетного менеджера pip;
pip install SpeechRecognition — установка библиотеки для распознавания человеческой речи;
pip install pyaudio — установка библиотеки для работы с микрофоном твоего компьютера.


MacOS:

pip install --upgrade pip — для обновления пакетного менеджера pip;
pip install SpeechRecognition — установка библиотеки для распознавания человеческой речи;
brew install portaudio — с помощью пакетного менеджера brew устанавливается необходимая для pyaudio библиотека;
pip install pyaudio — установка библиотеки для работы с микрофоном твоего компьютера.
